## 📌 Features
- 🛍️ Product management (CRUD operations for products)
- 📦 Order management
- 👥 User authentication and roles
- 💳 Payment integration (planned)
- 📊 Dashboard for analytics
- 📡 RESTful API for frontend interaction
- 🛠️ Admin panel for store management

## 🛠️ Technologies Used
- **Backend:** Laravel 11
- **Database:** SQLite
